社徽设计理念

1. 核心图形：XM 字母标识

标志中央的 "XM" 是 Xiangming（向明）的缩写，也是整个社团的视觉核心。
X 代表 Xiang（向）采用较强的几何切割结构，象征：无限可能（eXplore）
M 代表 Ming（明），采用渐变蓝色设计，像一条不断向上的数字轨迹。

2. 像素渐隐效果：代码世界的视觉语言

X 左侧由大量小方块逐渐消散：小方块代表计算机中的像素（Pixel）、数据单元（Data），从完整图形到数字碎片的过渡，表达"从基础代码与数据开始，通过创造形成完整的数字世界"。

3. 电路纹理：连接人与技术

背景中的线路元素来源于电子电路，象征计算机硬件基础。电路向外延伸，寓意学生的创造力从校园出发，连接更广阔的科技世界。

4. 二进制数字元素：计算机科学本源

背景中隐藏的 0 和 1 是计算机世界最基础的语言，代表二进制逻辑。同时采用低调透明的方式处理，不抢夺主体视觉，让科技元素成为"隐藏彩蛋"。

5. 外圈文字

上方中文"向明中学计算机社"，下方英文 Xiangming High School Programming Club。这个标志希望表达的不只是"计算机社"，而是一个由学生组成的科技创新共同体：从学习代码开始，培养逻辑思维；从解决问题开始，创造属于未来的数字世界。
